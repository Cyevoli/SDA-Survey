SDA Site Readiness
One-time setup to connect this form to your Microsoft account and OneNote.
Step 1 — Azure App Registration
Get your Client ID
1. Go to portal.azure.com and sign in
2. Search "App registrations" → New registration
3. Name it "SDA Survey" — leave defaults
4. Under "Authentication" → Add platform → Single-page app
5. Redirect URI: the URL of this file (or file:// if opened locally)
6. Copy the Application (client) ID from the Overview page
Step 2 — API Permissions (in Azure)
Add OneNote permissions
In your app registration → API permissions → Add permission → Microsoft Graph → Delegated:
Notes.Create and Notes.ReadWrite

Then click "Grant admin consent" if available, or your IT admin can do this.

Your Client ID is saved locally on this device only. Nothing is sent anywhere until you submit a form.
